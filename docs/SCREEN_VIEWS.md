# Screen Views

This gallery shows the main Mac Setup Buddy user flow. The UI adapts to the Mac's Light Mode or Dark Mode setting automatically.

## Welcome

The welcome screen introduces the setup process, shows the expected steps, and gives the user a clear start action.

![Welcome screen](screenshots/welcome.png)

## User Authentication

The authentication screen collects the user's work email address and prepares the app to continue with identity or enrollment tasks.

![User authentication screen](screenshots/authentication.png)

## Software Deployment

The deployment screen shows installation progress, current install state, system status, and a live activity stream.

![Software deployment screen](screenshots/software-deployment.png)

## Error Recovery

The recovery screen gives the user clear options when a policy or app installation times out.

![Error recovery screen](screenshots/error-recovery.png)

## Setup Complete

The completion screen confirms that setup finished and summarizes the user profile and device state.

![Setup complete screen](screenshots/setup-complete.png)
