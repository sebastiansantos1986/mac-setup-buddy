# Mac Setup Buddy Wiki

Mac Setup Buddy is a SwiftUI macOS setup assistant for guided device onboarding. It helps present a friendly setup flow while the Mac verifies the user, prepares the device, installs required apps, and confirms readiness.

## What It Includes

- Welcome and setup overview screen
- User authentication screen
- Software deployment progress dashboard
- Error recovery options
- Completion summary with user and device details
- Automatic Light Mode and Dark Mode theme support
